# Product+ive — User Manual

> **Last Updated:** 2026-04-01

---

## What is Product+ive?

Product+ive is your personal productivity coach. It teaches you 20 proven productivity, focus, learning, and study rules — and helps you actually practise them with guided sessions, smart task lists, and a Positivity Meter that grows as you build better habits.

## Getting Started

1. Open the app — you'll see 4 category cards on the Home screen
2. Tap any category to see the rules inside it
3. Tap a rule to learn about it and start a guided session
4. Complete sessions and tasks to grow your Positivity Meter!

## The 4 Categories

- 🟣 **Learning & Memory** — 1-4-7 Rule, Feynman Technique, 80/20 Rule, Spaced Repetition
- 🔵 **Focus & Sessions** — 20-20-20 Rule, Pomodoro Technique, 90-Minute Ultradian Rule
- 🟢 **Productivity** — 2-Minute Rule, 5-Second Rule, Eat the Frog, Parkinson's Law, 1-3-5 Rule
- 🟠 **Study Techniques** — Active Recall, Cornell Notes, Interleaving, Mind Mapping, SQ3R, Blurting, Elaborative Interrogation, Chunking

## Navigation

- **Home** (🏠) — Category grid, theme toggle, about button
- **+ Button** — Create a new task (optionally tag it with a rule)
- **Meter** (🧭) — Your Positivity Meter score, streak, and activity log

## Theme

Tap the sun/moon icon (top-right on Home) to switch between light and dark mode.

## Positivity Meter

Every action earns points:
- Start a rule session: +5
- Complete a timed session: +20
- Complete a task: +5 (+10 if rule-tagged)
- 7-day streak: +50
- First time using a rule: +25

Your meter resets weekly — fresh start every Monday!
